# 🌐 Dhanush Kumar — Developer Portfolio

Welcome to the repository of my developer portfolio! Designed as a high-performance, interactive, and beautifully animated single-page web app to showcase my projects, professional journey, and software engineering expertise.

---

## ⚡ Tech Stack

This site is powered by a modern, ultra-performant stack:

- **Framework**: [Next.js](https://nextjs.org/) (App Router, React 19)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [GSAP](https://gsap.com/) (GreenSock Animation Platform)
- **Components**: [shadcn/ui](https://ui.shadcn.com/) & [Lucide React](https://lucide.dev/)
- **Analytics**: Google Analytics

---

## 🛠️ Local Development

To run this project locally, clone the repository and execute the following commands:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the application:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser.
